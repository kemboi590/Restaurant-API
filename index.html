Overview

Welcome to the Restaurants API documentation. This API allows you to retrieve information about restaurants and cities with restaurants and more.

The restaurants API is designed to provide developers with a simple and efficient way to access our database of restaurant information. Whether you are building a restaurant review app, a city guide, or any other project, our API has you covered.

Please make sure to authenticate your requests using the API key provided to you during registration.

Endpoints

GET /restaurants

Retrieve a list of all restaurants.

GET /api/restaurants

GET /restaurants/:id

Retrieve details of a specific restaurant by ID.

GET /api/restaurants/:id

GET /citieswithrestaurants

Retrieve a list of cities that have restaurants.

GET /api/citieswithrestaurants

Examples

Get all restaurants

GET https://my-restaurant-api.azurewebsites.net/restaurants

Get a specific restaurant by ID

GET https://my-restaurant-api.azurewebsites.net/restaurants/{id}

Get all cities with restaurants

GET https://my-restaurant-api.azurewebsites.net/citieswithrestaurants
