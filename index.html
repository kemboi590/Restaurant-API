Overview

Welcome to the Restaurants API documentation. This API allows you to retrieve information about restaurants and cities with restaurants and more.

The restaurants API is designed to provide developers with a simple and efficient way to access our database of restaurant information. Whether you are building a restaurant review app, a city guide, or any other project, our API has you covered.

Please make sure to authenticate your requests using the API key provided to you during registration.
